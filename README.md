# Neural Networks from Scratch

## Overview
This project implements neural networks from scratch using Python and NumPy to understand the mechanics behind modern machine learning models.

The implementation includes:
- binary classification
- hyperparameter tuning
- regression
- multiclass classification
- comparison with scikit-learn

---

## Methods

### Binary Classification
- sigmoid activation
- binary cross-entropy loss
- gradient descent optimization

### Hyperparameter Tuning
Tested:
- number of iterations
- hidden layer size
- learning rate

### Regression
- modified output layer
- mean squared error loss
- evaluated using R²

### Multiclass Classification
- softmax activation
- extended prediction logic

---

## Results
- achieved ~99% accuracy on tuned classification tasks  
- demonstrated ability to learn nonlinear decision boundaries  
- achieved similar performance to scikit-learn models
